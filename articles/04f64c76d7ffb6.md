##### 原因を考える
中央値で非常に細かく補完しているだけど、機械学習モデル(今回使っているのはRandom Forest)から見ると、デメリットもあるらしい。
- **分布の歪み** ... 中央値(特定の値)が大量に注入されることで、データのバリエーションが失われ、モデルが「この値には特別な意味がある(実際は欠損なのに)」と誤学習した可能性。
- **「欠損であること自体」の情報消失** ... 年齢が分からない乗客には、何らかの傾向(例えば低所得層や特定の身分など)がある場合があって、補完してしまうことで「欠損である」という貴重なヒントを消してしまった可能性。

あくまで可能性の話で、それを検証する環境がないので深読みするのはここまでにして、対策を考えます。

### 欠損値補完3. Ageを補完せずに、欠損フラグを立てる。
この際、Ageの欠損値をそのままにして、欠損フラグを追加する実装で確認してみた。
```python
# 1. 欠損フラグを作成(欠損なら1、そうでなければ0)
df['Age_is_na'] = df['Age'].isnull().astype(int)
# 2. Age自体の欠損値は、モデルが扱えるように便宜上 -1 で埋める
# (Age_is_naフラグがあるため、モデルは「-1は特殊なデータ」と学習することを期待する)
df['Age'] = df['Age'].fillna(-1)
```

### 欠損値補完4. Ageをbin化(層別化)する。
bin化というのは、層別に分けるということですね。そもそもbinというのは箱って意味で、boxとの違いは、boxは"ただの箱"で、binは分類・仕分けをするための箱って意味らしい。
で、年齢を10歳ごとに区切って年代別にしてもいいのだけど、実データから分割点を調査して決めた方がよりスコアが上がる気がする。

まずは調査から。コードは下記。
```python
import pandas as pd

##### データ読み込み
train_data = pd.read_csv('/kaggle/input/competitions/titanic/train.csv') 
test_data  = pd.read_csv('/kaggle/input/competitions/titanic/test.csv')

import matplotlib.pyplot as plt

age_bin = pd.cut(train_data["Age"], bins=[0, 12, 17, 23, 24, 26, 27, 29, 30, 31, 32, 35, 36, 37, 38, 39, 40, 41, 50, 60, 100], right=False)
age_stats = train_data.groupby(age_bin)['Survived'].agg(['mean','count'])
print(age_stats)
overall_mean = train_data["Survived"].mean()

ax = age_stats['mean'].plot(kind='bar')
# 件数を上に表示
for lpct, val in enumerate(age_stats['count']):
    ax.text(lpct, age_stats['mean'].iloc[lpct], str(val), ha='center', va='bottom')
plt.axhline(overall_mean, color="red") # 全体の平均生存率
plt.xlabel("AgeBin")
plt.ylabel("Survival Rate")
plt.show()
```

### 欠損値補完5. AgeをRandom Forestで予測する。
[ココ](https://qiita.com/jun40vn/items/d8a1f71fae680589e05c)のサイトの方が、"欠損値がない完全なデータ（Pclass, Sex, SibSp, Parch)を使って、ランダムフォレストでAgeの欠損値を推定"する方法を試されてて結構効くって話なので、試してみます。
主要部分のコードは下記
```python
##### 全列の抽出＆数値化
age_pred_data = pd.get_dummies(all_data[['Age', 'Pclass', 'Sex', 'Fare', 'SibSp', 'Parch', 'Title', 'FamilySizeGroup']])

##### Ageがわかっているデータと欠損しているデータに分ける
age_known  = age_pred_data[age_pred_data['Age'].notnull()].values
age_unknown= age_pred_data[age_pred_data['Age'].isnull()].values

##### 学習用データ
X_age = age_known[:, 1:] # Age以外の特徴量
y_age = age_known[:, 0]  # Age(目的変数)

##### 回帰モデルで予測
rfr = RandomForestRegressor(random_state=1, n_estimators=100)
rfr.fit(X_age, y_age)

##### 欠損値の予測実行
predicted_ages = rfr.predict(age_unknown[:, 1:])

##### 元のall_dataに値を戻す
all_data.loc[all_data['Age'].isnull(), 'Age'] = predicted_ages
```
