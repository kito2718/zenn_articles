# Abstract
- 家族人数の特徴量エンジニアリングを追加。

# 概要
KaggleのTitanicコンペは、家族人数も考慮するとスコアアップするらしいです。考え方はシンプルで、乗客が「家族と一緒に乗っていたのか」「ひとりだったのか」の情報を特徴量にするって流れです。学習データには、`SibSp(兄弟・配偶者の人数)`と、`Parch(親・子供の人数)`が含まれるのでこれを使います。

## 1.「何人家族だったか？」を検証する。
何人家族だったかの情報が、生存率にどれぐらい効くかを確認します。
コードは下記になります。
```python
import pandas as pd

##### データ読み込み
train_data = pd.read_csv('/kaggle/input/competitions/titanic/train.csv') 
test_data  = pd.read_csv('/kaggle/input/competitions/titanic/test.csv')

####### 家族人数列生成処理 ここから
for df in [train_data, test_data]:
    # 家族人数列生成
    df['FamilySize'] = df['SibSp'] + df['Parch'] + 1

family_summary = train_data.groupby('FamilySize')['Survived'].agg(['mean', 'count'])
family_summary = family_summary.rename(columns={'mean': 'SurvivalRate', 'count': 'Count'})
####### 家族人数列生成処理 ここまで

display(family_summary)

###### グラフ表示
import matplotlib.pyplot as plt
fig, ax1 = plt.subplots()
# 生存率
bars = ax1.bar(family_summary.index, family_summary['SurvivalRate'])
ax1.set_ylabel('Survival Rate')
# 値表示
ax1.bar_label(bars, fmt='%.2f')
# サンプル数(折れ線)
ax2 = ax1.twinx()
ax2.plot(family_summary.index, family_summary['Count'], marker='o', color='red')
ax2.set_ylabel('Count')
# 値表示
for x, y in zip(family_summary.index, family_summary['Count']):
    ax2.text(x, y, str(y), ha='center', va='bottom')
# ラベル設定
plt.title('Survival Rate & Count by Family Size')
plt.show()
```
![](https://static.zenn.studio/user-upload/a8e5f0aa4543-20260502.png)
### 検証結果まとめ:
お1人様だと生存率は30%、家族人数2～4人が生存率が高くって、それ以上になるとまた生存率が低くなるという傾向が見えます。これはグループ化した方が効きそう。

### ⅰ.「何人家族だったか？」の情報を追加して提出。
「何人家族だったか？」の情報をデータ生成します。検証の結果から、下記が分かりました。
あきらかに、生存率は非線形で変化してるので、グループ化してOno-Hot Encodingにして、家族人数-生存率の相関関係を潰した方がいいですね。
- 1人 → 生存率悪し
- 2〜4 → 生存率良し
- 5〜7 → 生存率悪し
- 8+ → 生存率0

コードは下記。
```python

def family_size_group(size):

    df['FamilySize'] = df['SibSp'] + df['Parch'] + 1
```

## この章のまとめ
- 家族人数をグループ化すると効く。
- お1人様情報は、上記対応に含まれる。
- コードは下記
```python
import pandas as pd

##### データ読み込み
train_data = pd.read_csv('/kaggle/input/competitions/titanic/train.csv') 
test_data  = pd.read_csv('/kaggle/input/competitions/titanic/test.csv')

# 家族人数のグループ分け
def family_size_group(size):
    if size == 1:   return 'Alone'
    elif size <= 4: return 'Small'
    elif size <= 7: return 'Middle'
    else:           return 'Large'

####### One-Hot Encoding(敬称追加+家族人数)処理 ここから
for df in [train_data, test_data]:
    # 家族人数列 生成
    df['FamilySize'] = df['SibSp'] + df['Parch'] + 1
    # 家族人数をグループ分け
    df['FamilySizeGroup'] = df['FamilySize'].apply(family_size_group)
    # 敬称抽出→Title列生成
    df['Title'] = df['Name'].str.extract(' ([A-Za-z]+)\.', expand=False)
    # 敬称をまとめる
    df['Title'] = df['Title'].replace(['Mlle', 'Ms'], 'Miss')
    df['Title'] = df['Title'].replace('Mme', 'Mrs')
    df['Title'] = df['Title'].replace(['Capt', 'Col', 'Countess', 'Don', 'Jonkheer', 'Lady', 'Major', 'Sir'],'Rare')

##### trainとtestの全データを縦にくっつけて、`all_data`を取得する。
all_data = pd.concat([train_data, test_data], ignore_index=True)

##### One-Hot Encoding
all_data = pd.get_dummies(all_data, columns=['Title', 'FamilySizeGroup'])
##### 元に戻す
train_data = all_data.iloc[:len(train_data)].copy()
test_data  = all_data.iloc[len(train_data):].copy()
print(train_data.columns)
####### One-Hot Encoding(敬称追加+家族人数)処理 ここまで

##### 4つの特徴量を選択
features = ["Pclass", "Sex", "Age", "Fare"] \
         + [col for col in train_data.columns if "Title_" in col] \
         + [col for col in train_data.columns if "FamilySizeGroup_" in col]

##### 学習データを準備
X = train_data[features].copy()
y = train_data["Survived"]

X_test = test_data[features].copy()

##### 欠損値補完
age_median = X["Age"].median()
fare_median = X["Fare"].median()

X["Age"] = X["Age"].fillna(age_median)
X_test["Age"] = X_test["Age"].fillna(age_median)

X["Fare"] = X["Fare"].fillna(fare_median)
X_test["Fare"] = X_test["Fare"].fillna(fare_median)

##### Sex を数値化
X["Sex"] = X["Sex"].map({"male": 0, "female": 1})
X_test["Sex"] = X_test["Sex"].map({"male": 0, "female": 1})

##### モデル作成
from sklearn.ensemble import RandomForestClassifier
model = RandomForestClassifier(random_state=1)

##### 学習
model.fit(X, y)

##### 予測
predictions = model.predict(X_test)
predictions = predictions.astype(int)

##### 提出ファイル作成
output = pd.DataFrame({
    "PassengerId": test_data["PassengerId"],
    "Survived": predictions
})

##### 提出ファイル出力
output.to_csv("submission-4.csv", index=False)

##### 完了
print("submission.csv を作成しました 004")
```
